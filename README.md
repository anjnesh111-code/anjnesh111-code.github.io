# Data Science Portfolio Quest

Production-ready, gamified portfolio inspired by Codédex aesthetics using **React + Vite + Tailwind CSS + Framer Motion**.

## Project Structure

```text
portfolio-website/
├─ public/
│  ├─ favicon.svg
│  └─ icons.svg
├─ src/
│  ├─ components/
│  │  ├─ BadgesSection.jsx
│  │  ├─ Footer.jsx
│  │  ├─ HeroSection.jsx
│  │  ├─ MissionsSection.jsx
│  │  ├─ Navbar.jsx
│  │  ├─ SkillTreeSection.jsx
│  │  └─ StoryModeSection.jsx
│  ├─ data/
│  │  └─ portfolioData.js
│  ├─ App.jsx
│  ├─ index.css
│  └─ main.jsx
├─ index.html
├─ tailwind.config.js
├─ postcss.config.js
└─ vite.config.js
```

## Local Development

```bash
npm install
npm run dev
```

## Build Command

```bash
npm run build
```

Output folder: `dist`

## Vercel Deployment

1. Push this folder to a GitHub repository.
2. In Vercel, click **Add New Project** and import the repo.
3. Use these settings:
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Click **Deploy**.

## Branding Notes

- Dark fantasy neon dashboard style
- Gamified sections: Story Mode, Missions, Skill Tree, Badges
- Animated XP + level progression and unlock-style cards
